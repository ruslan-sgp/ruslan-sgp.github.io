NumPy Arrays

View vs Copy

  • View: A new object that looks at the same data in memory. Modifying the view changes the original array.
  • Copy: A completely independent object. Changing one does not affect the other.
import numpy as np

a = np.array([1, 2, 3])
view = a.view()
copy = a.copy()

view[0] = 99
print("Original after view change:", a)   # [99  2  3]

copy[1] = 88
print("Original after copy change:", a)   # [99  2  3] (unchanged)

Creating Arrays

From a list

np.array([1, 2, 3])

Special arrays

np.ones((2, 3))       # 2x3 array of ones
np.zeros((2, 3))      # 2x3 array of zeros
np.empty((2, 3))      # uninitialized values (fast, but random)
np.full((2, 3), 7)    # filled with 7
np.eye(3)             # identity matrix (3x3)

Ranges

np.arange(0, 10, 2)   # [0 2 4 6 8]
np.linspace(0, 1, 5)  # [0.   0.25 0.5  0.75 1. ]

Random arrays

np.random.rand(2, 2)       # uniform [0,1)
np.random.randint(1, 10, 5)  # 5 random ints 1–9

Shape and Reshape

a = np.arange(12)          # [0..11]
print(a.shape)             # (12,)

b = a.reshape(3, 4)        # reshape to 3x4
print(b)

Concatenation and Stacking

a = np.array([1, 2])
b = np.array([3, 4])

np.concatenate([a, b])          # [1 2 3 4]
np.stack([a, b])                # [[1 2]
                                #  [3 4]]
np.vstack([a, b])               # vertical stack
np.hstack([a, b])               # horizontal stack

Counting Frequencies

Using collections.Counter

from collections import Counter

data = [1, 2, 2, 3, 3, 3]
print(Counter(data))  # Counter({3: 3, 2: 2, 1: 1})

Using np.unique

arr = np.array([1, 2, 2, 3, 3, 3])
values, counts = np.unique(arr, return_counts=True)
print(values)  # [1 2 3]
print(counts)  # [1 2 3]

Using np.bincount

  • Works only with non-negative integers.
arr = np.array([1, 2, 2, 3, 3, 3])
print(np.bincount(arr))  # [0 1 2 3]
