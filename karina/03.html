1. Lists vs NumPy (np) vs Pandas (pd)

  • Lists: General-purpose, can hold mixed data types. Slower for large data.
  • NumPy Arrays (np.array): Homogeneous, optimized for numerical operations.
  • Pandas (pd): Built on NumPy, adds labeled data (Series, DataFrame) for structured analysis.
import numpy as np
import pandas as pd

# Python list
lst = [1, 2, 3]

# NumPy array
arr = np.array([1, 2, 3])

# Pandas Series
ser = pd.Series([1, 2, 3])

2. zip

  • Combines iterables element-wise.
names = ["Alice", "Bob"]
scores = [90, 85]
list(zip(names, scores))  
# → [('Alice', 90), ('Bob', 85)]

3. Loading and Understanding Data with Pandas

a) CSV

df = pd.read_csv("data.csv")  # load dataset

b) Series

  • 1D labeled array
pd.Series([10, 20, 30], index=["a", "b", "c"])

c) DataFrame

  • 2D table with rows & columns
data = {"Name": ["Alice", "Bob"], "Age": [25, 30]}
df = pd.DataFrame(data)

d) Head & Tail

df.head()   # first 5 rows
df.tail(3)  # last 3 rows

e) Info & Describe

df.info()       # column types, null counts
df.describe()   # summary stats (mean, std, min, etc.)

f) Columns

df.columns      # list of column names
df["Age"]       # access a single column
df[["Name","Age"]]  # multiple columns

g) Nulls

df.isnull().sum()   # count missing values
df.dropna()         # remove rows with nulls
df.fillna(0)        # replace nulls

h) Locating Rows

df.loc[0]      # by index label
df.iloc[1]     # by row position

i) Filtering

df[df["Age"] > 25]       # conditional filter
df[df["Name"] == "Bob"]  # match filter
