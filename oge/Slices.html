Основы срезов в Python

Срезы (slicing) — это мощный механизм в Python, который позволяет извлекать подпоследованияательности из списков, строк, кортежей и других последовательностей. Срезы используются для получения части коллекции без изменение оригинального объекта.

Синтаксис срезов

Синтаксис среза выглядит следующим образом:

последовательность[начало:конец]
  • начало — индекс, с которого начинается срез (включительно).
  • конец — индекс, до которого заканчивается срез (невключительно).

Примеры срезов

  1. Строка

    s = "Привет, мир!"
print(s[0:6])  # Вывод: Привет
print(s[8:11]) # Вывод: мир

  2. Список

    lst = [1, 2, 3, 4, 5, 6, 7, 8, 9]
print(lst[2:5])  # Вывод: [3, 4, 5]
print(lst[6:9])  # Вывод: [7, 8, 9]

  3. Кортеж

    t = (10, 20, 30, 40, 50)
print(t[1:4])  # Вывод: (20, 30, 40)

Особенности срезов

  • Опускание начального индекса

    s = "Привет, мир!"
print(s[:6])  # Вывод: Привет

  • Опускание конечного индекса

    s = "Привет, мир!"
print(s[8:])  # Вывод: мир!
