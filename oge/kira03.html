Урок 3

Что мы узнали

  1. Структура URL:

    • Составляющие URL:
      • Протокол (http:// или https://).
      • Имя хоста (доменное имя, например, hsc.edupoint.com.bd).
      • Путь к файлу (/ict/robotics.html).
    • Уровни домена:
      • Топ-уровень (.bd, .org, .ru).
      • Второй уровень (edupoint).
      • Третий уровень (hsc).

  2. Протоколы интернет:

    • HTTP и HTTPS - протокол передачи для передачи гипертекста (веб-страниц) и данных.
    • Разница между HTTP и HTTPS - защищённое соединение.
    • FTP - протокол для передачи файлов.

  3. Параметры в URL:

    • Как параметры влияют на содержимое страницы (например, ?q=пример).

  4. IP-адреса:

    • Формат IPv4 (4 байта, разделённые точками, макс. значение 255).
    • Пример: 192.168.1.101.

  5. Сетевые команды:

    • Команда ping используется для проверки доступности сервера.
    • ipconfig для просмотра сетевых настроек (IPv4, маска подсети, шлюз).

  6. Бинарный код и десятичные числа:

    • Перевод степеней двойки в десятичные значения (например, (2^0 = 1), (2^8 = 256)).

  7. Логические операции в поиске:

    • Использование & (И) и | (ИЛИ) для комбинирования запросов.
    • Пример: поиск по двум ключевым словам (Угол & Прямая).

  8. Решение задач с логикой:

    • Расшифровка скрытого адреса электронной почты из фрагментов.
    • Расчёт количества результатов поиска через диаграмму Венна.
