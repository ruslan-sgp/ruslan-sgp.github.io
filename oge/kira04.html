1. Почему Python?

Python — один из самых популярных языков программирования (на графике видно, что он лидирует на GitHub).

Его используют для:

  • ИИ и машинного обучения,
  • Научных вычислений,
  • Веб-разработки,
  • Автоматизации задач.

Официальный сайт Python

2. Как запустить Python?

Чтобы писать код на Python, нужны специальные программы или среды:

  • VS Code: универсальный редактор кода с поддержкой Python.
  • PyCharm: мощная среда разработки (IDE) специально для Python.
  • IDLE: простой редактор, который идёт вместе с Python.
  • Командная строка: можно запустить Python прямо в терминале (например, через py или python).

3. Что такое IDE?

IDE (Integrated Development Environment) — это программа, которая помогает писать код:

  • Integrated: объединяет редактор, компилятор, отладчик.
  • Development: помогает разрабатывать проекты.
  • Environment: создаёт удобное окружение для работы.

4. Основы синтаксиса Python

Присваивание переменных:

x = 5  # Записываем число 5 в переменную x
name = "John"  # Записываем строку "John" в переменную name

Вывод данных:

print("Hey")  # Печатаем текст "Hey" на экран
print(x)  # Печатаем значение переменной x (5)

Условия:

if hungry == True:  # Если переменная hungry равна True
    eat()  # Выполняем функцию eat()

Комментарии:

# Все, что после # — комментарий. Компьютер его игнорирует

5. Пример работы с Python в командной строке (REPL)

REPL — интерактивная оболочка, где можно быстро проверять код:

  • Read — прочитать команду.
  • Evaluate — выполнить её.
  • Print — вывести результат.
  • Loop — повторить цикл.

Примеры:

>>> 10 + 20  # Сложение
30
>>> print("Hello World")  # Вывод текста
Hello World
>>> "Hello " * 5  # Умножение строки
'Hello Hello Hello Hello Hello '
>>> 1024 > 15 * 35  # Сравнение чисел
True
