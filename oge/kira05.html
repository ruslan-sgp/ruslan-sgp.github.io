Пример программы: Периметр треугольника

Алгоритм:

  1. Получить длины трёх сторон (a, b, c).
  2. Проверить, можно ли построить треугольник:
    • c + b > a
    • a + c > b
    • a + b > c
  3. Если да — вывести периметр (p = a + b + c).
  4. Иначе — сообщить, что треугольник невозможен.
a = int(input("Введите длину первой стороны: "))
b = int(input("Введите длину второй стороны: "))
c = int(input("Введите длину третьей стороны: "))

x1 = c + b > a
x2 = a + c > b
x3 = a + b > c

if x1 and x2 and x3:
    p = a + b + c
    print("Периметр треугольника:", p)
else:
    print("Невозможно создать треугольник")

Типы данных в Python

  • Простые типы:

    • Целые числа (int): 10, -5.
    • Вещественные (float): 3.14, 0.5.
    • Булевы (bool): True, False.
    • Строки (str): "Hello", 'Python'.
    • None: Отсутствие значения.

  • Контейнеры:

    • Списки (list): [1, 2, 3].
    • Кортежи (tuple): (1, 2, 3) (неизменяемые).
    • Словари (dict): {"key": "value"}.
    • Множества (set): {1, 2, 3} (уникальные элементы).

Синтаксис Python

Синтаксис - правила написания кода.

  • Отступы: Важны для определения блоков кода (например, внутри условий).
  • Комментарии: Начинаются с #.
  • Форматирование строк: 
    print(f"Сумма: {a + b}")  # f-строки

Обработка ошибок

  • Пример: 
    >>> int("345")  # Корректно
345
>>> int("a")  # Ошибка!
Traceback ... ValueError: invalid literal for int() with base 10: 'a'
    В сообщении об ошибке обрати внимание на следующее:
    • тип ошибки (например, ValueError)
    • сообщение (например, invalid literal for int() with base 10: 'a')
    • место ошибки - файл, строка, и место в строке
  • Как исправить:
    Проверяйте тип данных перед преобразованием.
