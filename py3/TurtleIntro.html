Python Turtle

Сегодня познакомимся с Python Turtle для рисования. Этот урок основан на том, что ты уже знаешь, и откроет тебе новые идеи. Давай начнем!

Основные понятия Python Turtle

Вот некоторые важные концепции, используемые в коде сегодняшнего урока:

  1. Импорт модуля Turtle

    • import turtle: Импортирует модуль turtle. Это делает его доступным для нашей программы.
    • from turtle import *: Импортирует все функции из модуля turtle в текущий модуль, чтобы не использовать префикс turtle..

  2. Создание нескольких черепашек

    • t = turtle.Turtle(): Создает новую черепашку с именем t.
    • Ты можешь создать несколько черепашек (например, t1, t2). Каждая может иметь свои свойства.

  3. Базовые команды движения

    • forward(расстояние): Перемещает черепашку вперед на указанное расстояние.
    • backward(расстояние): Перемещает черепашку назад на указанное расстояние.
    • left(угол) и right(угол): Поворачивают черепашку влево или вправо на указанный угол.

    Эти команды действуют относительно текущего положения и направления черепашки.

  4. Настройка пера

    • color("название_цвета") или color(r, g, b): Устанавливает цвет пера.
    • width(размер): Изменяет толщину пера.

  5. Дополнительные команды движения

    • home(): Возвращает черепашку в исходную точку (0, 0) без изменения направления.
    • setheading(угол): Устанавливает направление черепашки на указанный угол.
    • goto(x, y): Перемещает черепашку в конкретную координату (x, y).

    Данные команды изменяют положение черепашки независимо от её текущего положения.

  6. Написание текста

    • write("текст"): Пишет текст на экране.
    • write("текст", font=('Arial', размер, 'normal')): Пишет текст указанным шрифтом.

  7. Отпечаток черепашки

    • stamp(): Оставляет копию формы черепашки в текущей позиции.

  8. Циклы для повторения

    • for i in range(n): Повторяет блок кода n раз. Широко используется для рисования узоров.

  9. Управление скоростью

    • speed(n): Устанавливает скорость рисования от 1 (медленно) до 10 (быстро); 0 - рисовать без задержки (самая быстрая скорость).

  10. Формы и размеры

    • shape("arrow"), shape("turtle"), и т.д.: Меняет внешний вид черепашки.
    • shapesize(stretch_wid, stretch_len, outline): Изменяет размер черепашки - длину, высоту, и контур (в относительных единицах, например 2, 3).

  11. Задержка и тайминг

    • delay(milliseconds): Приостанавливает программу на указанное время.

  12. Отслеживание позиции

    • pos(): Возвращает текущую позицию черепашки как (x, y) (кортеж!).

  13. Рисование сложных фигур

    • Используй циклы и углы для рисования многоугольников, зигзагов, звезд и других сложных узоров.

  14. Циклическая смена цветов

    • Используй списки цветов и арифметику (i % n) для динамической смены цветов.

Анализ кода

Разберем код пошагово:

Пример 1: Базовое движение с двумя черепашками 

import turtle
t = turtle.Turtle()
t.forward(100)
t.left(90)
t.color("blue")
t.width(5)
t.forward(200)

t1 = turtle.Turtle()
t1.color("red")
t1.forward(200)
t.right(45)
t.forward(150)

turtle.done()

Что здесь происходит:

  • Создание двух отдельных черепашек (t и t1) с разным цветом пера.
  • Базовые команды движения (forward, left, right).
  • Настройка пера (color, width).

Пример 2: Упрощенный код с использованием from turtle import *

from turtle import *
forward(100)
left(90)
color("blue")
width(5)
forward(200)
color("red")
forward(200)
right(45)
forward(150)
done()

Что здесь происходит:

  • Упрощает код за счет импорта всего содержимого из turtle.
  • Здесь у нас только одна черепашка, поэтому она меняет цвет на красный во время рисования.

Пример 3: Продвинутое движение и отпечатки

from turtle import *
speed(0)
width(5)
shapesize(3, 3, 1)
shape()
# stamp()
forward(500)
home()
left(135)
right(90)
setheading(45)
font14 = ('Arial', 14, 'normal')
write(100, font=font14)
angles = (0, 360, 45, 90, -90, 180)
for i, angle in enumerate(angles):
    home()
    delay(2000)
    setheading(angle)
    forward(100 + 25 * i)
    stamp()
    write(angle, font=font14)
color("red")
font16 = ('Arial', 16, 'normal')
write(pos(), font=font16)
goto(100, 100)
write(pos(), font=font16)
goto(-100, 100)
write(pos(), font=font16)
goto(-500, -500)
write(pos(), font=font16)
done()

Что здесь происходит:

  • Дополнительные команды движения: home (вернуться в центр) и setheading (установить угол поворота черепашки).
  • Написание текста с пользовательскими шрифтами.
  • Использование циклов для повторения действий (например, рисование линий под разными углами).
  • Отслеживание позиции с помощью pos().

Пример 4: Цветные круговые узоры

from turtle import *
speed(0)
colors = ['red', 'blue', 'green', 'yellow', 'purple', 'orange']
for i in range(120):
    color(colors[i % 6])
    setheading(i * 3)
    circle(200)
done()

Что здесь происходит:

  • Динамическая смена цветов с использованием списка и арифметики деления с остатком.
  • Рисование кругов с поворотом для создания узора.

Пример 5: Многоугольники, зигзаги и звезды

from turtle import *
shapesize(3, 5, 4)
speed(10)
width(5)
colors = ['red', 'blue', 'green', 'yellow', 'purple', 'orange']
# Пятиугольник
for i in range(6):
    forward(150)
    left(72)
# Зигзаг
for i in range(6):
    setheading(45)
    forward(50)
    setheading(-45)
    forward(50)
# Звезда
color("OrangeRed")
penup()
home()
pendown()

goto(100, 300)
goto(200, 0)
goto(-50, 250)
goto(250, 250)
home()
done()

Что здесь происходит:

  • Рисование многоугольников (например, пятиугольника) с помощью циклов и углов.
  • Создание зигзага при чередованием направления.
  • Рисование звезды путем соединения точек.

Интересные факты

  • Функция circle() также может рисовать дуги, если указать их протяженность (например, circle(100, 90) нарисует четверть круга).
  • Функция stamp() оставляет копию формы черепашки там, где она вызывается, что отлично подходит для создания узоров.
  • Остаток деления (i % n) — мощный инструмент для циклического прохождения списков, таких как цвета или углы.
