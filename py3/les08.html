Урок 8

Мы продолжаем рисовать городской пейзаж с помощью модуля turtle.

Код модуля coord.py

Вот усовершенствованный код модуля coord.py, который содержит функцию для рисования системы координат: 

from turtle import *


def draw_coordinates(size = 250, step = 50):
    """Draws a coordinate system with the given size."""
    initial_pos = pos()
    for i in range(-size, size + 1, step):
        # vertical lines
        penup()
        goto(i, -size)
        pendown()
        if i == 0:
            # main axis
            width(2)
            goto(i, size)
            setheading(90)
            stamp()
            width(1)
        else:
            goto(i, size)
        write(i)

        # horizontal lines
        penup()
        goto(-size, i)
        pendown()
        if i == 0:
            # main axis
            width(2)
            goto(size, i)
            setheading(0)
            stamp()
            width(1)
        else:
            goto(size, i)
        write(i)
    penup()
    goto(initial_pos)
    pendown()


if __name__ == "__main__":
    speed(0)
    draw_coordinates(500)
    done()

Объяснение

1. Импорт модуля в программу

Данный модуль уже содержит команду from turtle import *. Это значит, что все функции и переменные из этого модуля будут добавлены в текущий модель.

В нашей программе достаточно указать from coord import * в начале файла, чтобы использовать все функции и переменные из модуля coord.py, а также модуля turtle:

from coord import *

2. Функция draw_coordinates

def draw_coordinates(size=250, step=50):
    """Draws a coordinate system with the given size."""

Эта функция рисует систему координат. У нее есть два параметра:

  • size: размер сетки (по умолчанию 250).
  • step: расстояние между линиями сетки (по умолчанию 50).

3. Сохраняем начальную позицию черепашки

initial_pos = pos()

Функция pos() возвращает текущую позицию черепашки (координаты x и y). Мы сохраняем эту позицию в переменной initial_pos, чтобы потом вернуться туда после завершения рисования.

4. Цикл для рисования вертикальных линий

for i in range(-size, size + 1, step):
    # vertical lines
    penup()
    goto(i, -size)
    pendown()
    if i == 0:
        # main axis
        width(2)
        goto(i, size)
        setheading(90)
        stamp()
        width(1)
    else:
        goto(i, size)
    write(i)

Здесь мы рисуем вертикальные линии сетки:

  • range(-size, size + 1, step) создает последовательность чисел от -size до size с шагом step.
  • Для каждого значения i:
    1. Поднимаем карандаш (penup()) и перемещаемся в точку (i, -size) — начало вертикальной линии.
    2. Опускаем карандаш (pendown()) и рисуем линию до точки (i, size) — конца вертикальной линии.
    3. Если i == 0, то это главная вертикальная ось (ось Y). Мы делаем ее толще (width(2)) и добавляем стрелку (stamp()), указывающую направление.
    4. Пишем значение i рядом с линией (write(i)), чтобы пометить координаты.

5. Рисование горизонтальных линий

# horizontal lines
penup()
goto(-size, i)
pendown()
if i == 0:
    # main axis
    width(2)
    goto(size, i)
    setheading(0)
    stamp()
    width(1)
else:
    goto(size, i)
write(i)

Аналогично предыдущему блоку, здесь мы рисуем горизонтальные линии:

  • Начинаем с точки (-size, i) и рисуем линию до (size, i).
  • Если i == 0, то это главная горизонтальная ось (ось X). Мы делаем ее толще и добавляем стрелку.
  • Пишем значение i рядом с линией.

6. Возвращение в начальную позицию

penup()
goto(initial_pos)
pendown()

После завершения рисования, мы поднимаем перо, возвращаемся в начальную позицию (initial_pos) и снова опускаем перо.

7. Проверка запущенного модуля

if __name__ == "__main__":
    speed(0)
    draw_coordinates(500)
    done()

Здесь проверяем, запущен ли файл напрямую (главная программа - main) или импортируется как модуль. __name__ - это специальная переменная в Python, которая равна "__main__" при запуске файла напрямую, или содержит имя модуля при импорте.

Если значение переменной __name__ равно __main__, значит файл запущен в качестве главной программы, то есть нужно нарисовать только сетку координат и оставить окно октрытым. В этом случае выполняем следующие команды:

  • speed(0) устанавливает максимальную скорость черепашки.
  • draw_coordinates(500) вызывает функцию рисования координат с размером сетки 500.
  • done() останавливает программу, чтобы окно не закрылось сразу после выполнения.
