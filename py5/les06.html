Урок 6

Исследуем на практике гибкость Python при работе с данными и некоторые ошибки, которые могут возникнуть.

Вот некоторые идеи урока:

1. Повторение последовательностей

[0] * 10  # Создает список из 10 нулей
'a' * 10  # Создает строку из 10 букв 'a'
  • Показывает работу оператора * с последовательностями

2. Исследование модулей

import random
dir(random)  # Выводит все атрибуты/методы модуля random
  • Демонстрирует инспекцию модулей с помощью dir()

3. Типичные ошибки с модулями

random(10)  # Ошибка: модуль нельзя вызывать
random.random(10)  # Ошибка: random() не принимает аргументы
  • Показывает частые ошибки при работе с модулями и функциями

4. Генерация случайных чисел

random.random()  # Число от 0.0 до 1.0
random.random() * 24 - 12  # Масштабирование в [-12, 12)
random.randint(-12, 12)  # Целое число в диапазоне
  • Различные техники генерации случайных чисел

5. Операции со списками

l.index(2)  # Находит первое вхождение
"Hello World".index('o')  # Работает и со строками
"Hello World".index('o', 5)  # Поиск с начальной позиции
  • Поиск элементов в последовательностях

6. Функция max() и её вариации

max('Hello World')  # 'r' (макс. символ по ASCII)
max([1,2,3])  # 3
max(a, b, 30, 40, 50)  # Работает с несколькими аргументами
  • Гибкость функции max() с разными аргументами

7. Словари и списковое включение

[name for name in PD]  # Получает ключи
[name for name in PD if PD[name] == 2]  # Фильтрация по значению
  • Итерация и фильтрация словарей

8. Конкатенация списков

[1, 23, 45] + [1,5]  # Объединение списков
l + l  # Дублирование списка
  • Объединение последовательностей через +

9. Обновление словарей

PD.update({"Иванов": 3, "Сидоров": 5})  # Правильно
PD + PD  # Ошибка: нельзя складывать словари
  • Правильные и неправильные способы изменения словарей

10. Генераторы списков с условиями

[x**2 for x in l]  # Квадраты всех элементов
[x**2 for x in l if x != 0]  # Квадраты ненулевых элементов
  • Фильтрация и преобразование элементов

Выводы:

  1. Перегрузка операторов (* и + работают по-разному для разных типов)
  2. Важность правильных аргументов функций
  3. Способы работы с последовательностями
  4. Различия в поведении словарей и списков
  5. Отладка ошибок через эксперименты

l = []

    l.append(x)

Задание 4.5

PD = {
    "Кузьмин": 5,
    "Круглов": 5,
    "Кашина": 5,
    "Лобова": 3,
    "Лукин": 2,
    "Озеров": 2,
    "Орехов": 3,
}

l = []
for name in PD:
    mark = PD[name]
    if mark == 2:
        l.append(name)
print("Неуспевающие студенты:")
print(l)

# вариант со списковым включением
print([name for name in PD if PD[name] == 2])
