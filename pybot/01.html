Исходный код

Мы написали следующий код. Вставьте токен вашего бота:

# bot.py

# импорт модулей telebot
import telebot
from telebot.types import Message

# создать бот и указать токен
bot = telebot.TeleBot('здесь укажите ваш токен')

# создать функцию для обработки сообщений
@bot.message_handler()
def handle_message(message: Message):
    print(message)

# начать опрос сервера
bot.polling()

Примечание: @bot.message_handler() - это декоратор, - особенность языка Python. Здесь декоратор используется, чтобы указать, что telebot должен вызывать функцию handle_message при поступлении входящих сообщений.

💡 Обратите внимание: вызов bot.polling() должен быть после объявления функции handle_message(), иначе опрос начнётся до создания функции-обработчика сообщений, и она не будет работать.
